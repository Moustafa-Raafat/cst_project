Apply CSS to HTML

CSS can be applied to HTML document as:

  1. Inline style
  2. Internal style sheet
  3. External stylesheet file

Inline Style

An inline style may be used to apply a unique style for a single element.

Internal Style Sheet

Internal style sheet may be used if one single page has a unique style.

Syntax

The CSS syntax rule is made up of three parts:

  1. selector
  2. property
  3. value

Selectors

IDs Selector

The ID attribute is used to define a unique style for an element

Class Selector

Classes allow you to define a style which can be applied to multiple elements on your page.
